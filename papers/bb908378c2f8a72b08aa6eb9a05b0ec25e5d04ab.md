# Complexity Beyond the Trigram: Identifying Sign Languages from Video Using Neural Networks

**Year:** [[2016]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** P. Toman

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/bb908378c2f8a72b08aa6eb9a05b0ec25e5d04ab)

## TL;DR

SLANG-3k is released, a benchmark three-class dataset for sign language identiﬁcation on the internet and it is established that a shallow convolutional neural network with only a sixth of a second of temporal information can outperform a deep network without temporal information in identifying the language in which a person is signing.

# An Overview of the SiGML Notation and SiGMLSigning Software System

**Year:** [[2004]]
**Topics:** [[topics/computer-science|Computer Science]]
**Authors:** R. Elliott; J. Glauert; V. Jennings; J. R. Kennaway

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/752dc97dc4d4483564cadc5f153082ecee2507e7)

## TL;DR

An overview of the SiGML notation, an XML application developed to support the definition of Sign Language sequences for performance by a computer-generated virtual human, or avatar, is presented.

# Sign Language Production using Neural Machine Translation and Generative Adversarial Networks

## TL;DR

A novel approach to automatic Sign Language Production using state-of-the-art Neural Machine Translation (NMT) and Image Generation techniques, capable of producing sign videos from spoken language sentences.

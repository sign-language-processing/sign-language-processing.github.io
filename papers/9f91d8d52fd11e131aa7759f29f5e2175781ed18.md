# Exploring Visual Representations for Sign Language Translation

**Year:** [[2022]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Maram A. Mohamed; Xavier Giró; Laia Tarrés

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/9f91d8d52fd11e131aa7759f29f5e2175781ed18)

## TL;DR

This work investigates the impact of using different types of visual sign language representation for SLT using the state-of-the-art Sign Language Transformers model, and compares the translation output performance of two types of body pose estimation models as the authors' skeleton extractor, and 2D CNN features trained on the test dataset.

# Stack Transformer Based Spatial-Temporal Attention Model for Dynamic Multi-Culture Sign Language Recognition

**Year:** [[2025]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Koki Hirooka; Abu Saleh Musa Miah; Tatsuya Murakami; Yuto Akiba; Yong Seok Hwang; Jungpil Shin

## TL;DR

This paper proposes a Stack Spatial-Temporal Transformer Network that leverages multi-head attention mechanisms to capture both spatial and temporal dependencies with hierarchical features using the Stack Transfer concept and demonstrates improved performance in tasks such as gesture recognition and human activity prediction.

# LSE-Sign: A lexical database for Spanish Sign Language

**Year:** [[2015]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/medicine|Medicine]], [[topics/linguistics|Linguistics]]

## TL;DR

The LSE-Sign database is a free online tool for selecting Spanish Sign Language stimulus materials to be used in experiments and contains 2,400 individual signs taken from a recent standardized LSE dictionary, and a further 2,700 related nonsigns.

# PeruSIL: A Framework to Build a Continuous Peruvian Sign Language Interpretation Dataset

**Year:** [[2022]]
**Topics:** [[topics/medicine|Medicine]], [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** G. Bejarano; Joe Huamani-Malca; Francisco Cerna-Herrera; Fernando Alva-Manchego; Pablo Rivas

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/bce54ad9184fec6b5ad3787855ebf94ca1f0af64)
- [ACL Anthology](https://aclanthology.org/2022.signlang-1.1)

## TL;DR

A framework for creating a multi-modal sign language interpretation dataset based on videos is presented and it is used to create the first dataset for Peruvian Sign Language (LSP) interpretation annotated by hearing volunteers who have intermediate knowledge of PSL guided by the video audio.

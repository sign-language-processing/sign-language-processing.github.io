# Towards Automatic Sign Language Annotation for the ELAN Tool

**Year:** [[2008]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Philippe Dreuw; H. Ney

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/f5666a7984516c6c86281d5e96a7b4c8798f461d)

## TL;DR

Preliminary results show that the performance of a statistical machine translation improves using automatically generated annotations, and the proposed framework provides easy access to the output of an automatic sign language recognition and translation framework.

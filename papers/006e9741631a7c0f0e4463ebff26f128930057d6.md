# Tools for Language: Patterned Iconicity in Sign Language Nouns and Verbs

**Year:** [[2015]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/medicine|Medicine]], [[topics/linguistics|Linguistics]]
**Authors:** C. Padden; So-One Hwang; R. Lepic; Sharon Seegers

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/006e9741631a7c0f0e4463ebff26f128930057d6)
- [DOI](https://doi.org/10.1111/tops.12121)

## TL;DR

It is found that ASL signers generally favor the instrument strategy when naming tools, but when describing tools being used by an actor, they are significantly more likely to use more handling forms.

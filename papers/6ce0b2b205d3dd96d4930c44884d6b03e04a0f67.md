# Bidirectional Skeleton-Based Isolated Sign Recognition using Graph Convolutional Networks

**Year:** [[2022]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Konstantinos M. Dafnis; Evgenia Chroni; C. Neidle; Dimitris N. Metaxas

## Links

- [ACL Anthology](https://aclanthology.org/2022.lrec-1.797)

## TL;DR

This work proposes a new skeleton-based method that involves explicit detection of the start and end frames of signs, trained on the ASLLVD dataset, and employs a bidirectional learning approach within a Graph Convolutional Network (GCN) framework.

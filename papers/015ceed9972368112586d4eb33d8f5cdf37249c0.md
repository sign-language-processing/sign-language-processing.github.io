# EnSenias: Technological tool to learn, teach, improve and use Panamanian Sign Language

**Year:** [[2020]]

## TL;DR

The results of the analysis show that EnSenias is necessary and suitable to learn the sign language, provides universal accesibility, is useful to learn and teach sign vocabulary, and serves as an elementary dictionary and translator in sign language.

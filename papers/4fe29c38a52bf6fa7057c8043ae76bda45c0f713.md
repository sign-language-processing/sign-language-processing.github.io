# A Machine Learning-based Segmentation Approach for Measuring Similarity between Sign Languages

**Year:** [[2022]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Tonni Das Jui; G. Bejarano; Pablo Rivas

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/4fe29c38a52bf6fa7057c8043ae76bda45c0f713)
- [ACL Anthology](https://aclanthology.org/2022.signlang-1.15)

## TL;DR

This work proposes to support the similarity measurement between sign languages through a video-segmentation-based machine learning model that will quantify this match among signs of different countries’ sign languages using a machine learning approach, and finds that the intermediate-strict metric coincides with a more classical analysis of the similarity between British and American Sign Language, as well as with the classical low measurement between Indian and British sign languages.

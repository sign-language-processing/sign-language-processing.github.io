# Large Lexicon Project: American Sign Language Video Corpus and Sign Language Indexing/Retrieval Algorithms

**Year:** [[2010]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** V. Athitsos; C. Neidle; S. Sclaroff; Joan P. Nash; Alexandra Stefan; Ashwin Thangali; Haijing Wang; Quan Yuan

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/329faa0fe6c1b433110423bd4b28306edc3c0624)

## TL;DR

Progress is described in an ongoing project to build a system that helps users look up the meaning of ASL signs by building an application that lets the user submit a video of a sign as a query, and presents to the user the most similar signs from the system database.

# An Efficient Sign Language Recognition (SLR) System Using Camshift Tracker and Hidden Markov Model (HMM)

**Year:** [[2021]]
**Topics:** [[topics/computer-science|Computer Science]]
**Authors:** P. Roy; Pradeep Kumar; Byung-Gyu Kim

## TL;DR

A novel end-to-end SLR system from RGB video-sequences that identifies double and single hand gestures is proposed and new features, from skin region and hand trajectories, are proposed to improve the gesture classification performance.

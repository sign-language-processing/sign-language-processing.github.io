# A Dataset for Irish sign language recognition

**Year:** [[2017]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Marlon Oliveira; Houssem Chatbri; Ylva Ferstl; M. Farouk; S. Little; N. O’Connor; Alistair Sutherland

## Links

## TL;DR

A new image dataset for Irish Sign Language (ISL) recognition is introduced and experiments using Principal Component Analysis (PCA) where the dataset reached 95% recognition accuracy are reported.

# iLex - A tool for Sign Language Lexicography and Corpus Analysis

**Year:** [[2002]]
**Topics:** [[topics/computer-science|Computer Science]], [[topics/linguistics|Linguistics]]
**Authors:** Thomas Hanke

## Links
- [Semantic Scholar](https://www.semanticscholar.org/paper/c4355c046b452729ac4f5dbdbe1729c4ffe55ed2)
- [ACL Anthology](https://aclanthology.org/L02-1330)

## TL;DR

A tool that combines features found in empirical sign language lexicography and in sign language discourse transcription, which supports the user in lexicon building while working on the transcription of a corpus.
